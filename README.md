API Endpoints

- All in Postman

- GET/api >
  Returns the entire list of objects in the data storage
- GET/api/:id >
  Returns a single object that matches the given ID
- POST/api >
  Add a new object to the data storage
- PUT/api/:id >
  Updates an existing object with the data in the request body. Works for numeric or string IDs.
- DELETE/api/:id > Removes the object that matches the given ID from the data storage.
  Also shows a message when deleted.

- Add PORT from Postman
  Use: http://localhost:3000/
